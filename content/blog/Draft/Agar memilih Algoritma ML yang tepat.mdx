Regression Algorithm digunakan untuk memprediksi nilai dari sebuah variabel target berdasarkan nilai dari beberapa variabel input atau fitur. Model regresi linier relatif sederhana dan memberikan rumus matematika yang mudah ditafsirkan untuk menghasilkan prediksi. Algoritma ini digunakan untuk memprediksi nilai di masa depan berdasarkan nilai historis, seperti memprediksi penjualan produk, prediksi harga saham.

Bisa dipakai untuk: Forecasting, Predictions, Process Optimization.

Regularization Algorithm adalah teknik yang Â digunakan untuk mengendalikan overfitting pada model. Overfitting terjadi ketika model data terlatih dengan sangat baik sehingga ia "menghafal" data tersebut, tetapi tidak dapat melakukan prediksi yang baik pada data yang belum pernah dilihat sebelumnya. Dalam situasi ini, regularization membantu model untuk menjadi lebih umum dan mencegahnya terlalu fokus pada detail-detail kecil dari data pelatihan. Biasanya digunakan dalam feature selection. Algoritma Lasso (Least Absolute Shrinkage and Selection Operator) digunakan untuk mengendalikan kompleksitas model dengan mengurangi beberapa koefisien model menjadi tepat nol. Fitur yang memiliki koefisien nol dianggap tidak berpengaruh pada model. Dengan begitu, kita dapat menghilangkan fitur yang tidak diperlukan dalam model dan meningkatkan kinerja prediksi.

Rule system (Association Rule Learning Algorithm) digunakan untuk menemukan hubungan antara variabel dalam database besar.

Instance-Based Algorithm bekerja dengan membandingkan data yang ingin diprediksi dengan data yang telah dilihat sebelumnya untuk menemukan kemiripan, dan kemudian membuat prediksi berdasarkan kesamaan tersebut.

Bisa dipakai untuk: Prediction, Detection.

Neural Network Algorithm adalah teknik machine learning yang memproses data input dengan beberapa lapisan untuk mengenali hubungan antara sekumpulan data melalui proses yang meniru cara kerja otak manusia.

Bisa dipakai untuk: Computer vision, Generated system, Prediction, Approximation, Recommendation.

Dimensionality Reduction Algorithm: digunakan ketika ingin menghilang informasi yang redundan, noise dan outliers dari data. Contoh: Analisis Teks.

Bisa untuk : Big data visualization,

Decision Tree Algorithms memiliki struktur seperti flowchart, ini adalah algoritma yang membangun model prediksi atau klasifikasi berdasarkan serangkaian aturan keputusan yang dapat diinterpretasikan dengan mudah. Contoh: Menentukan kategori risiko kredit pelanggan

Bisa dipakai untuk: Sentiment Analysis, Prediction

Clustering Algorithms adalah metode dalam tipe unsupervised learning yang digunakan untuk mengelompokkan titik data berdasarkan kesamaan tertentu. Pengelompokan ini dilakukan untuk menemukan kelompok-kelompok alami dalam data tanpa tanpa memerlukan label atau kelas pada data. Contoh: klaster daerah Covid-19.

Bisa dipakai untuk: Recommend Systems, Targetted Marketing, Customer Segmentation, Optimization

Bayesian Algorithm biasanya digunakan dengan cara membangun model probabilistik dari terjadinya suatu peristiwa berdasarkan data yang dimiliki, dapat disebut [probabilistic classifiers](https://towardsdatascience.com/introduction-to-probabilistic-classification-a-machine-learning-perspective-b4776b469453). Contoh: deteksi email spam.

Bisa dipakai untuk: Prediction, Classification.

Ensamble Algorithm: teknik yang menggabungkan beberapa model machine learning untuk memperbaiki akurasi dan performa prediksi.

Contoh penggunaan ensemble algorithm adalah pada prediksi kinerja akademik mahasiswa, dimana dapat digunakan beberapa algoritma machine learning seperti Decision Tree, Random Forest, dan Gradient Boosting. Dengan menggabungkan hasil prediksi dari beberapa algoritma tersebut, maka dapat meningkatkan akurasi prediksi kinerja akademik mahasiswa dalam sistem pembelajaran.

Bisa dipakai untuk: Robot Navigation, Game AI

Deep Learning Algorithm: digunakan ketika kita ingin menyelesaikan masalah yang kompleks dan tidak bisa dipecahkan dengan menggunakan metode konvensional atau machine learning tradisional. Algoritma deep learning memiliki kemampuan untuk belajar dari data yang kompleks dan dapat menghasilkan prediksi yang lebih akurat. Namun, karena kompleksitasnya, penggunaan deep learning memerlukan sumber daya komputasi yang besar dan data yang cukup banyak. Juga dikenal sebagai deep neural learning atau deep neural network.

Bisa dipakai untuk: Prediction, Detection, Efficiency

Contohnya adalah pengenalan wajah, deteksi objek, pengenalan suara, dan pemrosesan bahasa alami. Selain itu, deep learning juga digunakan untuk analisis data besar, seperti analisis genom dan prediksi pergerakan pasar.

Time Series Algorithm adalah metode forecasting yang digunakan untuk memprediksi data yang diurutkan dalam periode waktu tertentu

Bisa dipakai untuk: Forecasting, Prediction
